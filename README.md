# Simple Static Website

This is a basic static website built with **HTML** and **CSS** only.

## Features
- Clean and simple layout
- Responsive design
- Beginner friendly
- No frameworks or JavaScript

## Project Structure
- index.html
- style.css
- README.md

## How to Run
Open index.html in your browser or open the folder in VS Code.
